# Parent Guide

- Document owner: Product and Operations
- Last reviewed: 2026-03-24
- Primary use: Parent-facing guide for child tracking and safety visibility

## What You Can Do Today

- Sign in to the parent portal.
- View children linked to your account.
- Open live route tracking for a child with an assigned route.
- Review current route position through map polling.

## Typical Parent Workflow

1. Sign in to the parent portal.
2. Review the child cards on the dashboard.
3. Select a child and open the live map.
4. Watch the current bus position update.

## What the Current Portal Does Not Fully Support Yet

- push notifications for alerts or boarding and alighting events
- notification history or inbox review
- absence reporting
- explicit consent and notification preference management
- rich ETA and next-stop detail

## Troubleshooting for Parents

- If live location looks stale, the route may not have new GPS data yet.
- If a child does not appear, check that the account is linked to the correct student record.
- If you expected a proactive alert, note that notification delivery is still an active gap in the current platform.

## Privacy Notes

- Parent access is intended to be limited to linked children and their route context.
- Tracking data should be viewed only for legitimate student-care and transport needs.
