# SBTM Public Release Business Plan

**Document Owner**: Product & Business Development
**Last Updated**: 2026-04-30
**Status**: Implementation Phase

## Overview

This directory contains the complete business plan and implementation strategy for launching SBTM as a public, open-source school bus management platform with a dual-license commercial model.

## Business Model

- **Revenue Strategy**: Dual-license (OSS deployment tools + commercial support)
- **Target Audience**: School districts (direct B2G sales)
- **Source Code Strategy**: Deployment automation open source (MIT), source code viewable under commercial license
- **Support Model**: Community-first (GitHub), with paid support tiers
- **Timeline**: 4-6 week fast-track launch
