# Module 2 - Parent App (Web)

## Status
Implemented as a demo-first web app with mock data.

## Location
- `apps/parent-app/web`

## Tech Stack
- React 19 + Vite
- TailwindCSS
- React Router + Leaflet

## Functionality
- Mock login (any credentials)
- Child cards with status
- Live map with simulated SSE updates
