# Parent App

This module contains the Parent Application, which allows parents to track their child's bus and receive notifications.

## Components

### Web (`/web`)
A React + TypeScript application built with Vite.
- **Port**: 5173 (Default)
- **Run**: `cd web && npm run dev`
- **Test**: `cd web && npm test`

### Mobile (`/mobile`)
A Flutter application (To be implemented).

## Features
- Login (Mocked)
- Dashboard showing children status
- Live Map Tracking (Simulated)
